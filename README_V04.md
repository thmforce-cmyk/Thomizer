# Thomizer hardware GUI v0.4

Updated compact test build.

Changes:
- Corrected branding to THOMIZER.
- Made top modules significantly lower and more compact.
- Moved control values into consistent inline positions.
- Reduced THOMLAY AY-Y420 / HOW MUCH panel height.
- Softened FORCER so the delay tail does not choke immediately at low values.
- Re-tuned HOW MUCH for softer glue / tape compression.
- Made player dock full-width and fixed to the bottom/side edges.
- Changed lock behavior to avoid fixed-body scroll jumps.
- Added stronger scale-dependent accent/glow theme changes.

Note: iOS Safari browser chrome cannot be fully suppressed while testing in Safari. For no address bar, launch the PWA from the iOS Home Screen.
